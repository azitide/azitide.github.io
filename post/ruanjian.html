HUO

电脑开机的第一步：应该是等待自启动的软件与插件启动完成，这个时间都够我背一个”两只老虎爱跳舞，一只没有…咳咳”,说一下，火绒安全软件里面的安全工具非常好用

管理自启动

那么我们用火绒安全软件里面的安全工具—-启动项管理，就调整不需要的软件。

弹窗管理

还有一些软件会弹窗一些xx广告出来，真的是”摸黑进厕所–找si“啊，在安全工具—-弹窗拦截，就可以关闭所有软件弹窗。

官网下载
火绒安全 (huorong.cn)

生产力

截图翻译

我们在学习、工作时，大多时候都会是英文优先级很高，毕竟世界百分之90%的文献都是英文的，我们有时候难免看不懂，手机翻译就很累赘了，拍不清还翻译错了，为了防止”爆率是真的高“，我们最好还是用本机翻译，打开网易有道翻译–设置–快捷键位，就可以进行截图了

官网下载
网易有道翻译(youdao.com)

截图

截图，市面上的载图软件参差不齐各有各有的特点，Q、微信那些又要登录真的麻烦，我一想用插件多好啊，又小又能自启动，又不用登录！

一款名为「Snipaste」的截图软件，体积仅有 12M 。

功能：快捷键自定义大小截图，固定截图在屏幕，截图上进行标注

使用官网
Snipaste - 截图 + 贴图

推荐下载：在微软商店直接搜名字，点击安装即可

截图识字

这个东西有好有坏，因为字体太多识别错字的几率也挺大的，我用过了很多的截图软件，如功能比较全的开源软件shareX，但是识别字体的正确率也比较低，我现在比较中意的还是微软官方出品了一个插件功能包增强版，识别到的字体正确率是我用过最高的—-Power Toys ！

Power Toys 设置–文本提取器–快捷键设置

官网：
Microsoft PowerToys | Microsoft Learn

下载方法：在微软商店直接搜Power Toys，点击安装即可

游览器适配

游览器插件的启用就是扩展最猛的功能，chrome、火狐相同的Chromium内核。尤其是强大的同步和扩展功能，能让你的浏览器瞬间开挂！

下面的插件适配：

  • Edge游览器

  • 谷歌游览器

  • 火狐游览器

好用：

  • AdGuard 广告拦截器

AdGuard 广告拦截器可有效的拦截所有网页上的所有类型的广告，甚至是在 Facebook，Youtube 以及其它网站的广告！

  • Convertio 万能格式转化器

在线将文件从一种格式转换为另一种格式。支持超过 2500 种不同的转换:

  • IDM下载器(IDM Integration Module)

IDM不限速，最大支持32线程下载。支持网页视频、音乐、文件下载、自动捕获链接、自动识别文件名、批量下载、计划下载任务、站点抓取等。配和IDM软件使用。

  • Global Speed: 视频速度控制

与几乎所有视频和音频流媒体站点兼容，包括Youtube，Netflix，哔哩哔哩，腾讯视频，百度网盘， 爱奇艺等。自定义播放速度，最大支持16倍倍速。

插件推荐网站：Chrome插件扩展程序_扩展迷 (extfans.com)

下载方法：
Edeg打开下方网址点击–获取–下载好后，打开拓展-点击启用

Microsoft Edge Addons
